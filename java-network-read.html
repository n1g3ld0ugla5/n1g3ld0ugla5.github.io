Java Process Class File Download (Log4J)



Important Notice:

Not compatible with Java 17+, which uses read() syscalls

Description

A critical vulnerability has been found in log4j, a widely-used open-source utility used to generate logs inside java applications.
The vulnerability CVE-2021-44228, also known as Log4Shell, permits a Remote Code Execution (RCE), allowing the attackers to execute arbitrary code on the host.
